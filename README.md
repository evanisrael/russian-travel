# Проектная работа: "Путешествие по России"

Проект курса "Веб-разработка" от Яндекс.Практикума по тематике "Адаптивная верстка"

### Используемые технологии
* HTML и CSS
* Flexbox
* Grid Layout
* @media
* БЭМ Nested

**Описание проекта**

Проект рассказывает о возможности путешествия по России.
